# Introduction

This script is designed to convert molecular dynamics (MD) trajectories generated by GPUMD into a single unit-cell configuration. A detailed methodology for visualizing atomic distributions from such simulations is described in [**National Science Review 11, nwae216 (2024)**](https://doi.org/10.1093/nsr/nwae216).

# Input Files

Please prepare the following two input files:

1. **`inp.dat`**: specifies the supercell size, number of atomic types, and other basic settings;
2. **`dump.xyz`**: contains the MD trajectory coordinates.

We have included a complete example for Cs<sub>3</sub>Bi<sub>2</sub>I<sub>6</sub>Cl<sub>3</sub>, with inp.dat, dump.xyz, and all supporting files.
